Предпосылки:
Библиотека настроена и работает.
В системе реализованы методы:
GetAvailableBooks()
BorrowBook(userId, bookId)
GetBooksBorrowedByUser(userId)
Есть тестовый пользователь и тестовые книги.

Тест: Добавление и извлечение доступных книг (Test_AddAndRetrieveAvailableBooks)
Шаги:
Инициализировать библиотеку с двумя книгами: "Война и мир" (ID=1) и "Преступление и наказание" (ID=2).
Вызвать метод GetAvailableBooks().
Ожидаемый результат:
Метод возвращает список из двух доступных книг.
Количество книг: 2.

2. Тест: Заимствование книги уменьшает количество доступных книг (Test_BorrowBook_ReducesAvailableBooks)
Шаги:
Пользователь с ID=1 заимствует книгу с ID=1 ("Война и мир").
Вызвать метод GetAvailableBooks().
Ожидаемый результат:
Количество доступных книг: 1.
Оставшаяся книга: "Преступление и наказание" (ID=2).

3. Тест: Получение списка заимствованных книг для пользователя (Test_GetBorrowedBooksByUser)
Шаги:
Пользователь с ID=1 заимствует книгу с ID=1 ("Война и мир").
Вызвать метод GetBooksBorrowedByUser(1).
Ожидаемый результат:
Список заимствованных книг содержит 1 книгу.
ID книги: 1 ("Война и мир").

4. Тест: Заимствование несуществующей книги вызывает исключение (Test_BorrowNonExistentBook_ThrowsException)
Шаги:
Попытаться заимствовать книгу с несуществующим ID=999 для пользователя с ID=1.
Ожидаемый результат:
Метод BorrowBook(1, 999) выбрасывает исключение с сообщением: "Book not found.".

5. Тест: Получение заимствованных книг для несуществующего пользователя вызывает исключение (Test_GetBooksForNonExistentUser_ThrowsException)
Шаги:
Попытаться вызвать метод GetBooksBorrowedByUser(999) для пользователя с ID=999, которого нет в системе.
Ожидаемый результат:
Метод выбрасывает исключение с сообщением: "User not found."
